CSS文字控制属性
2026-05-29 20:27:14 # CSS #html #css文字控制属性

1.字体大小font-size

单位必须用px，不能没有单位，不然不生效

2.字体粗细font-weight

属性值可以用数字和关键字
数字（开发使用）不能加单位px
正常：400
加粗：700
关键字
正常：normal
加粗：bold

3.字体倾斜font-style

清除文字默认的倾斜效果
属性值：
正常：normal
倾斜：italic

4.行高line-height

属性值两种：
数字+px
数字（当前标签font-size属性值的倍数
行高是上间距+文本高度+下间距=行高
行高测量方法：一行文字的最顶端到下一行文字的最顶端
垂直居中技巧：行高和盒子高度一致

5.字体族font-family

属性值可以书写多个字体名，各个字体名用逗号隔开，执行顺序从左往右依次查找
属性最后设置一个字体族名，网页开发建议使用无衬线字体

6.字体复合属性font

属性的简写方式，一个属性对应多个值的写法，各个属性值用空格隔开
必须按顺序写:
font：是否倾斜 是否加粗 字号/行高 字体
字号和字体值必须书写，否则font属性不生效

7.文本缩进text-indent

行内元素（特点不独占一行）img，em，input，label，a,span,strong等无效
必须是块级元素或者行内块，且有文本内容，只对第一行文本缩进
属性值：
数字+px
数字+em（推荐：1em=当前标签字号大小）

8.文本对齐text-align

本质是控制内容的对齐方式，属性要设置给内容的父级,对其内的行内及行内块元素生效

9.修饰线text-decoration

可删除修饰线，添加修饰线，下划线，删除线等，如删除超链接的下划线
属性值 效果
none 无
underline 下划线
line-through 删除线
overline 上划线

10.颜色color

color: rgba(red, green, blue, alpha);

颜色表示方式 属性值 说明 使用场景
颜色关键字 颜色英文单词 red，green 学习测试
rgb表示法 rgb(r ,g,b) rgb表示红绿蓝，取值0-255 了解
rgba表示法 rgba（r，g，b，a） a表示透明度，取值0-1 开发使用，实现透明色
十六进制表示法 #rrggbb #000000，#ffcc00，简写#000，#fc0 开发使用（从设计稿复制）

颜色关键字 颜色英文单词 red，green 学习测试

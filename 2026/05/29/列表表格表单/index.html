1、列表

作用：布局排列整齐的区域
分类：

无序列表（unordered list）：不需要规定顺序

​

ul嵌套li，ul是无序列表，li是列表条目且独占一行
ul里面只能包裹li标签，li里面可包含任何内容

有序列表（ordered list）：需要规定顺序

​与无线列表相同，ol只能包裹li标签

定义列表（Definition list）：

dl嵌套dt（Definition term）和dd（Definition Description），dl（Definition list）是定义列表，dt是定义列表的标题，dd是定义列表的描述或详情

​dl里面只能放dt和dd，dt和dd可以包含任何

2、表格

1.基本用法

​用来展示数据
标签：table嵌套tr，tr嵌套td/th

标签名 说明
table 表格
tr 行
th 表头单元格（加粗居中）
td 内容单元格

​表格默认没有边框线，使用border属性可以为表格添加边框线
表格会根据内容变化大小

2.结构标签

​显示不会有变化，但是可以告诉浏览器结构标签，了解即可

标签名 含义 特殊说明
thead 表格头部 表格头部内容
tbody 表格主体 主要内容区域
tfoot 表格底部 汇总信息区域

3.合并单元格

​将多个单元格合并成一个，以合并同类信息
可跨行和跨列合并
（1）明确合并目标
（2）保留最左最上的单元格，添加属性（取值是数字，表示需要合并的单元格数量）
跨行合并，保留最上单元格，添加属性rowspan
跨列合并，保留最左单元格，添加属性colspan
（3）删除其他单元格

3、表单

作用：收集用户信息
使用场景：登录,注册,搜索
input标签基本使用（单标签）
不独占一行
input标签type属性值不同，则功能不同
<input type="...">

属性值 说明
text 文本框，用于输入单行文本
password 密码框
radio 单选框
checkbox 多选框
file 上传文件

4、占位文本placeholder属性

提示信息
文本框密码框都可使用
<input type="text" placeholder="提示信息">

5、实现单选框radio

使用name属性实现单选功能

属性名 作用 特殊说明
name 控件名称 控件分组，同组只能选中一个（单选功能）
checked 默认选中 属性名和属性值相同，布尔属性，写一个单词

6、上传文件file

默认只能上传一个文件，添加multiple属性可以实现文件多选功能
<input type="file" multiple>

7、多选框-checkbox

多选框也叫复选框
默认选中：checked
<input type="file" checked>

8、下拉菜单select

select嵌套option，select是下拉菜单整体，option是下拉菜单的每一项

name=”” id=”” value=””未来发送数据的时候用，不用可先删除

9、文本域

作用：多行输入文本的表单控件
<textarea name="" id="" cols="30" rows="10">默认文本</textarea>
name=”” id=””上传数据用
cols=”30” rows=”10”设计尺寸用，但一般用css设计

10、label标签

作用：网页中，某个标签的说明文本
用label标签绑定文字和表单控件的关系，增大表单控件的点击范围
增大范围写法
写法1：

​label标签只包裹内容，不包裹表单控件
设置label标签的for属性值和表单控件的ID属性值相同
写法2：

​label标签直接包裹内容和表单控件，不需要写属性

11、按钮-button（双标签）

type属性值：

属性值 说明
submit 提交按钮，点击后可以提交数据到后台（默认功能）
reset 重置按钮，点击后将表单控件恢复默认值
button 普通按钮，默认没有功能，一般配合JavaScript使用

12、form标签

表单区域标签，其他标签要放到里面才能管理
action属性值是发送数据地址
<form action=""></form>

13、无语义的布局标签

作用：布局网页（划分网页区域，摆放内容）
div（双标签）：独占一行
span（双标签）：不换行

14、字符实体

作用：在网页中显示预留字符

显示结果 描述 实体名称
空格 &nbsp；
< 小于号 &lt；
> 大于号 &gt;

JS-函数
2026-05-29 20:02:58 # JS #Js-函数

函数是可以执行特定任务的代码块，可以实现代码复用，简化代码

1.函数的声明

function 函数名（）{
函数体
}

2.命名规范

尽量小驼峰
前缀尽量为动词
命名建议：常用动词约定

3.关键字：

有特殊含义的字符，如let、var、if、for等

4.函数传参

function （传递的参数列表）{

}
1.多个参数用逗号隔开
2.声明时括号里的参数为行参，调用时的实际参数为实参
3.return后面代码不在执行，return后面的数据不要换行写
4.有return后，外面可以调用结果
5.return只能返回一个值，返回多个值时用中括号返回
6.结束函数用return，break结束循环，return结束函数

5.作用域

1.全局作用域：作用于所有代码执行的环境（整个script标签内部，或一个独立的js文件），全局有效，全局声明的变量为全局变量
2.局部作用域：函数内的代码环境就是局部作用域，也称函数作用域，局部有效，局部内为局部变量

（1）函数内部没有声明，直接赋值的变量也当全局变量
（2）形参可以看做函数的局部变量
（3）不同作用域的相同变量名的变量不冲突
（4）采取就近原则确定变量最终值

6.匿名函数

function（）{}
无法直接使用
使用方法：
1.函数表达式
将函数赋值给变量，通过变量名调用，及变量名当函数名调用
let fn = function(){
代码
}
具名函数的调用可以写在任何位置，匿名函数只能写在声明后，必须先声明，后调用

7.立即执行函数

避免全局变量直接污染

要加分号结束,最后的括号相当于调用函数
另一种写法：(function（）{}())
也是第一个括号形参，第二个括号实参

8.逻辑中断

||,当前面为真时，中断，后面的不在执行；都为假，以最后为准
n=4||0 n=4; n=false||0 n=0

&&，当前面为假时，中断，后面不在执行；都为真，以最后为准
n=4&&0 n=0; n=false&&0 n=false

9.转换布尔值

··、0、null、false、NaN、undefined转换为布尔值后为假，其余为true

10.类型转换

1.引号+是拼接
2.减号只能用于数字，他会将空字符串··转换为0
3.null转换为数字变为0
4.undefined转换为数字变为NaN
5.undefined和任何操作为NaN
NaN和任何操作为NaN

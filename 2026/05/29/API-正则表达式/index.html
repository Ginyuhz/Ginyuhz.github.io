API-正则表达式
2026-05-29 21:00:25 # API #正则表达式

1.介绍

是一种匹配模式，也是对象
作用：
表单验证
过来敏感词
提取文字

2.语法

1.定义规则

const 变量名 = /表达式/,不需要引号，想查找什么写什么

2.查找

规则变量名.test(被检查的字符串)，返回t/f
规则变量名.exec(被检查的字符串),返回数组，否则为null

3.元字符

1.边界符

表示位置，开头和结尾，必须用什么开头，用什么结尾
^表示以谁开始，$表示以谁结束
^ha表示以ha开头
^ha$表示精确匹配，只有ha为true

2.量词（表示重复次数）

设定某个模式出现的次数

量词 含义 示范
* 重复0次或更多次 ab*，正确：a、ab、abb、abbb，错误：ac（a后面不是b）
+ 重复一次或更多次 ab+，正确：ab、abb、abbb，错误：a（必须有一次）
？ 重复0次或1次 ab？，正确：a、ab，错误：abb（只有0或一次）
{n} 重复n次，必须是n次 ab{3}，正确：abbb，错误：abb（只能是3次）
{n,} 必须重复n次或更多次 ab{3，}，正确：abbb，abbbbb，错误：abb（只能是3次或更多次）
{n,m} 必须重复n到m次，不能有空格 ab{3，5}，正确：abbb，abbbbb，错误：abb（只能是3到5次）

重复离最近的

3.字符类（比如/d表示0~9）

[]匹配字符集合，只要包括其中一个都可以，精确匹配就只选一个
[^a]取反，除a以外都行

4.预定义

大写相当于取反
\d 相当于[0-9]
\D相当于[^0-9]
\w相当于[A-Za-z0-9_]
\W相当于[^A-Za-z0-9_]
\s匹配空格（包括换行符，制表符、空格符等）相当于[\t\r\n\v\f]
\S相当于

4.修饰符

语法：
/表达式/修饰符
i不区分大小写
g全局匹配符合的表达式
替换符合的内容：字符串.replace(/正则表达式/ig,’替换的文本’)

5.change事件

内容发生改变在触发
6.input的属性required：非空

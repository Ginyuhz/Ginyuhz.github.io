1、DOM

DOM是浏览器提供的一套专门用来操作网页内容的功能

2.document.querySelector(‘选择器’)

获取匹配的第一个元素document.querySelector(‘选择器’),一定要加引号
括号里面的选择器写法和css一样，可以直接操作
获取的元素是js里的对象

3.document.querySelectorAll(‘选择器’)

获取匹配的对象集合，是伪数组，没有数组的方法
要得到里面的每一个对象，要用for遍历

4.修改对象内容

获取文字内容：对象.innerText，显示纯文本，不解析标签
修改文字内容：对象.innerText=新值
innerHTML解析标签，与innerText类似，但是识别标签

5.修改样式属性 对象.style.样式属性 = ‘值’，要单位

1.如果样式有-连接，则用小驼峰命名法，如backgroundImg
2.body可以直接用document.body
3.操作类名（className）操作css
对象.className = ‘active’ //active是个css类名
会覆盖前面的类，所以用
对象.className = ‘active nav’ //nav是原有css类名
4.classList
追加类：对象.classLIst.add(‘类名’),类名不加点，并且是字符串
切换类：对象.classLIst.toggle（’类名’）,有就删除，没有就加上+
删除类：对象.classLIst.remove（’类名’），类名不加点，并且是字符串
查看类：对象.classLIst.contains（’类名’）,判断是否包含类，有返回true，没有返回false

6.获取表单的值

表单.value
const input = document.querySelector(‘input’)
input.type=’password’

7.只接收布尔值

复选框input.checked=true 选中
禁用按钮disabled，默认false，不禁用

8.自定义属性

data-，只要这个开头就是自定义属性
获取
对象.dataset：获取全部自定义属性集合
对象.dataset.自定义属性：获取指定自定义属性值

9、定时器（间歇函数）

开启定时器

setInterval（函数，间隔时间（单位毫秒））
每隔一段时间调用函数
有名函数不要加小括号，会直接调用，而不是间隔调用

关闭定时器

定时器返回的是一个id数字
let n = setInterval（函数，间隔时间（单位毫秒））
clearInterval（n）

AJAX-axios、http协议、form-serialize插件
2026-06-09 15:48:27 # AJAX #axios #http协议 #form-serialize插件

介绍

  1. AJAX是可以让浏览器和服务器进行数据通信的技术
  2. Axios是HTTP请求库，专门发GET/POST等网络请求（调后端接口），底层基于Promise封装
  3. 什么是HTTP请求库：封装好的工具，调用后端接口和服务器收发数据的JS工具包，不需要自己进行额外的操作
  4. Axios = 给原生又老又繁琐的 AJAX（XHR）套了一层「便利外壳」
    把所有重复、底层、易错的代码全部封装，只留给你最简单的调用方式，同时加上 Promise 和一堆实用功能。
    Axios 一共包了 8 大块：
    1. 隐藏原生  XMLHttpRequest  底层代码
    2. 自动 JSON 解析
    3. 转为 Promise 异步写法（支持 then / async await）
    4. 简化 GET/POST 等请求方法
    5. 自动处理请求参数、请求头
    6. 统一网络错误、状态码判断
    7. 增加拦截器、全局配置
    8. 拓展超时、取消请求、进度监听等功能

2、URL

http://hmajax.itheima.net/api/province
组成：

协议(http://)+域名(hmajax.itheima.net)+资源路径(/api/province)

1.协议规定数据传输格式
2.域名表示访问哪台服务器
3.资源路径标识在该服务器的路径

4、常用请求方法

请求方法：对服务器资源要执行的操作
method：请求方法,（GET可省略，默认的）
data：提交数据
两种写法：
语法：

请求方法 作用 写法1 写法2
get 获取数据 axios.get() method:get
post 提交数据，提交一条全新数据 axios.post() method:post
put 修改数据，必须传完整对象，不传字段清空 axios.put() method:put
delete 删除数据 axios.delete() method:delete
patch 修改单个数据，其余字段保留 axios.patch() method:patch

如put要修改用户数据中的头像，必须将整条用户数据都上传，不然只传头像，用户数据中的名字，年龄等数据就好清空
patch只需要提交头像就可以，数据会更改头像，保留其他字段

5、Http协议

1.请求报文

请求报文：浏览器按照协议要求的格式，发送给服务器的内容
组成：请求行，请求头，空行，请求体
请求报文可以进行错误排除

2.响应报文

服务器按照协议格式要求，返回给浏览器的内容
组成：响应行，响应头，空行，响应体
响应状态码：用来表明是否成功完成
1xx 信息
2xx 成功
3xx 重定向信息
4xx 客户端错误
5xx 服务端错误

6、接口文档

7、form-serialize插件

作用：快速收集表单元素值
函数语法：

hash设置数据结构
取值true，数据为js对象
false，数据为查询字符串
empty：设置是否获取空值
true：获取空值
flase：不获取
保存的格式是将输入框的name作为键值保存

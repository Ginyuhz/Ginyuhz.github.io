API-事件
2026-05-29 20:26:38 # API # 事件 #事件监听 #事件类型 #事件 #冒泡

1、事件监听

三要素：事件源（谁被触发）、事件类型、事件处理程序
元素对象.addEventListener(‘事件类型’,要执行的函数)

2、事件类型（要加引号）：

鼠标事件：
点击click
经过mouseenter
离开mouseleave
鼠标移动：mousemove

焦点事件：
获得焦点focus
失去焦点blur

属性选择器：
[]，可以通过属性选择器控制自定义属性

键盘事件：
键盘按下keydown
键盘弹起keyup

文本事件
文本输入input

change事件
当内容发生改变时触发

3、事件对象

在事件监听回调的函数的第一个参数就是事件对象，常见命名是e、event等，e相当于浏览器给你的事件详情，e.target可以获取当前的元素

常用属性：

type：
获取当前事件类型

clientX/clientY：
获取光标相对于浏览器可见窗口左上角的位置

OffsetX/offsetY:
获取光标相对于当前DOM元素左上角的位置

pageX/pageY:
获取光标相对于整个html页面左上角的位置，包含页面滚动的距离,相对于clientX+页面水平滚动距离

key（键盘事件）:
获取用户按下的键盘的值，当按下a时，e.key就是“a”
现在不提倡使用keyCode，因为他返回的时数字编码（如回车是13）

trim（）方法：去除字符串左右的空格

4、环境对象

this表示当前运行的环境
每个函数都有环境对象，普通函数里（如fn（））this指向的是window
谁调用，this就是谁

5、回调函数

如果将A函数作为参数传递给B，那么A就是回调函数

6、全选

checked：选中值为true，未选为flase

7、事件流

指事件完整执行的完整经过

1.两个阶段：

元素对象.addEventListener(‘事件类型’,要执行的函数，是否使用捕获机制)
是否使用捕获机制 = true 代表触发，flase代表冒泡，默认flase
事件捕获：父到子，外到里
事件冒泡：一定发生，子到父，里到外，只冒泡同名事件

2.阻止冒泡

事件对象.stopPropagation()
阻止元素默认行为：e.preventDefault（）

3.解绑事件

on事件方式，直接用null覆盖就可以实现事件的解绑
解绑：removeaddEventListener（）

8、事件委托

给父级绑定事件
事件对象.target当前点到的元素
事件对象.target.tagName可以获得真正触发的元素，要写大写

9、其他事件

1.页面加载事件

1.load
window.addEventListener(‘load’,fuction())等待页面所有资源加载完毕，就回去执行回调函数
window也可以写其他资源

2.DOMContentLoaded
document.addEventListener(‘DOMContentLoaded’,fuction())等待html标签加载完毕，无需等待样式表，图片加载完成

2.页面滚动事件

scroll：监听整个页面滚动
属性：scrollLeft、scrollTop（被卷去的头部大小）、scrollTo（x,y）

3.获取html元素documentElement

document. documentElement.scrollTop可以获取元素移动距离，返回为数字，不带单位，单位为像素

4.页面尺寸事件resize

浏览器窗口大小发生变化时触发的事件
获取宽高：
clientWidth/clientHeiht(不包含边框，margin，滚动条等)

10、元素尺寸于位置

获取元素自身宽高（包含padding，margin）：offsetWidth和offsetHeight
获取元素距离自己‘定位父级元素’的左上距离：offsetLeft/offsetTop（只读）
返回元素大小及相对于(视口)的位置：element.getBoundingClientRect(),当页面滚动时，获取的位置也会发生改变
属性：scroll-behavior:smooth;平滑滚动

html基础命令及操作
2026-05-29 02:56:22 # Html #html

1、HTML基本骨架

​html标签里的内容是整个网页
head标签里用来存放给浏览器看的信息，如css
body标签里的是网页主体，给用户看的
title标签在head里面，是网页标题

2、注释

在要注释那行的任意一个位置按Ctrl加斜杠
变成注释浏览器不显示
取消注释仍然是Ctrl加斜杠

3、标题标签h（双标签）

h1到h6字号文字加粗逐渐减小，且独占一行
h1一个网页只用一次，是一级标题，用来方标题和logo

4、段落标签p（双标签）

​独占一行，段落间有空隙
不管标签里的内容是否分段，在同一标签都显示成一段

5、换行<br>与水平线标签<hr>(单标签)

​不需要包裹内容的就是单标签
浏览器不识别代码当中enter键换行，要使用
换行

6、文本格式化标签

​一般使用左边标签，具有强调意义，显示没有区别
strong 加粗 b
em 倾斜 i
ins 下划线 u
del 删除线 s

7、图像标签<img>单标签

（1）src 用于指定图像的位置和名称

<img src ='图像的url'>
输入./可以在当前路径寻找图片，vscode有提示功能
且图片在同一行

（2）图片属性

alt 替换文本 图片无法显示的时候显示的文字
title 提示文本 鼠标悬停在图片上面时显示的文字
width 图片宽度 值为数字，没有单位
heigh 图片高度 值为数字，没有单位
浏览器是等比例缩放
src alt title等是属性名，其后是属性值，属性写在尖括号理标签名后，标签名和属性之间用空格隔开

8、路径

​路径指的是从起点到终点经历的路线

路径分类

相对路径：

从当前文件位置出发查找目标文件
找自己电脑内的文件，一般使用相对路径

绝对路径：从盘符出发查找文件

Windows从盘符出发，默认是反斜线(),但是斜线也可以用
Mac从根目录（/）出发
文件不在自己电脑，复制在线网址

/表示进入某个文件夹里面，格式是 文件名/
./表示当前文件所在的文件夹
..表示上一级文件， ../表示进入上一级，../../表示进入上上一级

9、超链接a（双标签）

​a标签包含的内容是可以点击跳转的内容，可以是文字，也可以是图片
属性target=”_blank”在新窗口跳转页面
href属性值是跳转地址，是必须属性，开发初期不知道超链接跳转地址，属性值写#，表示空链接，不会跳转
跳转到本地文件：相对路径查找

10、音频标签audio（双标签）

​ <audio src="音频的URL"></audio>

属性 作用 特殊说明
src（必须属性） 音频URL 支持格式：MP3 Ogg Wav
controls 显示音频控制面板 是布尔型属性，只要出现就表示true，不写则该属性=false，所有HTML布尔属性都遵循这个规则
loop 循环播放
autoplay 自动播放 为了提示用户体验，浏览器一般会禁止使用自动播放功能

11、视频标签video

<video src="视频的URL"></video>

属性 作用 特殊说明
src（必须属性） 视频URL 支持格式：MP4 Ogg WeBM
controls 显示音频控制面板 是布尔型属性，只要出现就表示true，不写则该属性=false，所有HTML布尔属性都遵循这个规则
loop 循环播放
muted 静音播放
autoplay 自动播放 为了提示用户体验，浏览器支持在静音状态下自动播放功能
